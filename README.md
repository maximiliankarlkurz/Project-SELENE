# Project-SELENE
Project SELENE: Das Projekt umfasst Delta-v-Berechnungen und die Modellierung eines Roboterarms. Fokus liegt auf der Anwendung kinetischer und kinematischer Konzepte, detaillierter Herleitung und numerischen Bestimmung von Drehmomenten und Leistung. Praxisnaher Beleg mechanischer Grundlagen zur Prüfungsvorbereitung.
